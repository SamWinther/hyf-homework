World Trading Data is an API for catching the real time price of shares at stock exchange

This API needs a paid API-Key but for first 250 request, one can have a free API-Key.

It returns a Promise. It means that to get the value, the developer should use async/await function.

To test the API, you need share symbol. For example, the symbol of Novo Nordisk is NVO.
